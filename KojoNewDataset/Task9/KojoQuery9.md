Draw two semicircles connected by a straight vertical line. The first semicircle bulges to the right, and the second bulges to the left. The flat (diameter) side of each semicircle lies along the vertical line, which runs the full height of both diameters combined.

`semicircle_radius` specifies the radius of each semicircle in pixels — the diameter of each semicircle, and therefore the length of each half of the vertical line, is twice this value. (`semicircle_radius = 70`)

Write short, simple Kojo code.
