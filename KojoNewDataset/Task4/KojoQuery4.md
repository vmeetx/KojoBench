Draw a regular pentagon. The bottom side is horizontal, so the shape sits flat with one vertex pointing straight up. The turtle starts at one of the bottom corners.

`pentagon_side` specifies the length of each side of the pentagon in pixels. (`pentagon_side = 200`)

All five sides are equal. Use `setHeading(36)` before the drawing loop so the bottom side ends up horizontal. Write short, simple Kojo code using a loop.
