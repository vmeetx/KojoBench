Draw an equilateral triangle. The base is a horizontal line, and the single vertex points straight up. The turtle starts at the bottom-left corner of the triangle.

`triangle_side` specifies the length of each side of the triangle in pixels. (`triangle_side = 200`)

All three sides are equal. Use `setHeading(60)` before the drawing loop so the base ends up horizontal at the bottom. Write short, simple Kojo code using a loop.
